Install

安装和激活 Install and activate

当前支持 Chrome，按单设备授权。 Currently supports Chrome with single-device authorization.

1
下载 `asm-assistant-0.7.43.zip` 并解压。 Download `asm-assistant-0.7.43.zip` and unzip it.
2
打开 Chrome 扩展管理页，开启开发者模式。 Open Chrome extensions and enable Developer mode.
3
点击“加载已解压的扩展程序”，选择目录。 Click “Load unpacked” and select the folder.
4
打开插件，输入激活码。 Open the extension and enter the activation code.
5
如果要用订单同步，先准备飞书多维表链接和飞书授权码。 If you need order sync, prepare the Feishu table link and Feishu auth code first.
6
第一次先打开 Etsy Sold Orders 页面，同步一次当前订单页完成店铺绑定。 Open Etsy Sold Orders and sync the current page once to bind the shop for the first time.
7
绑定完成后，再填写飞书表链接和飞书授权码。 After the shop is bound, fill in the Feishu table link and auth code.
FAQ FAQ

常见问题 Common questions

先把最常见的疑问说清。 These are the most common questions before using it.

会自动发送吗？ Will it auto-send?

不会，你来发。 No, you send it yourself.

怎么授权？ How does activation work?

一机一码。 One device, one code.

谁能用？ Who can use it?

内部和符合条件的学员。 Internal users and eligible students.

订单同步还要准备什么？ What else is needed for order sync?

除了插件激活码，还要准备飞书多维表链接和飞书授权码。第一次要先在 Etsy Sold Orders 页面同步一次当前订单页，完成店铺绑定后再填飞书配置。 Besides the extension activation code, you also need the Feishu table link and auth code. The first step is syncing the current Etsy Sold Orders page once to bind the shop, then fill in the Feishu settings.
